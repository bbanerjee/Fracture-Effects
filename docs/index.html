Parsim : Particle-based Simulation

About ParSim

ParSim is a fork of the Uintah simulation software with a focus on solid mechanics modeling and simulation.

Current development is focussed on Vaango, a MPM code, and ParaEllip3D, a DEM-SPH-Peridynamics code.

We are also exploring the development of a user interface for these tools.
