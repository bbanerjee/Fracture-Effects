Building Visit --> Visit 2.5.2

  • Create directories in which to build VisIt:
    mkdir $HOME/visit
mkdir $HOME/visit/visit2.5.2
mkdir $HOME/visit/visit2.5.2/build
mkdir $HOME/visit/visit2.5.2/prebuild

  • Get the VisIt sources, build script and prerequisites by executing the following commands:
     cd $HOME/visit
  wget http://www.sci.utah.edu/devbuilds/icse/visit/2.5.2/visit2.5.2.tar.gz
  cd visit2.5.2
  wget http://www.sci.utah.edu/devbuilds/icse/visit/2.5.2/patch2.5.2.diff
  cd prebuild
  wget http://www.sci.utah.edu/devbuilds/icse/visit/2.5.2/build_visit2_5_2.txt
  wget http://www.sci.utah.edu/devbuilds/icse/visit/2.5.2/_prereqs/Imaging-1.1.6.tar.gz
  wget http://www.sci.utah.edu/devbuilds/icse/visit/2.5.2/_prereqs/IceT-1-0-0.tar.gz
  wget http://www.sci.utah.edu/devbuilds/icse/visit/2.5.2/_prereqs/MesaLib-7.8.2.tar.gz
  wget http://www.sci.utah.edu/devbuilds/icse/visit/2.5.2/_prereqs/Python-2.6.4.tgz
  wget http://www.sci.utah.edu/devbuilds/icse/visit/2.5.2/_prereqs/cmake-2.8.8.tar.gz
  wget http://www.sci.utah.edu/devbuilds/icse/visit/2.5.2/_prereqs/pyparsing-1.5.2.tar.gz
  wget http://www.sci.utah.edu/devbuilds/icse/visit/2.5.2/_prereqs/pyside-combined-1.0.7.tar.gz
  wget http://www.sci.utah.edu/devbuilds/icse/visit/2.5.2/_prereqs/qt-everywhere-opensource-src-4.7.4.tar.gz
  wget http://www.sci.utah.edu/devbuilds/icse/visit/2.5.2/_prereqs/visit-vtk-5.8.tar.gz

  • Make the build script executable:
    chmod u+x build_visit2_5_2.txt

  • Set up some environment variables
    export PAR_COMPILER=/usr/bin/mpic++
  • Look for Xinput.h in your system. If you do not have that installed you will also need libxi.
    sudo apt-get install libxi-dev
  • In Ubuntu 13.10 you will need to go to visit2.5.2/src/common and add TARGET_LINK_LIBRARIES for libpthread and libdl to the visitcommon library.
  • Start the VisIt prebuild
    ./build_visit2_5_2.txt --makeflags -j4 --no-visit --parallel --slivr --python-module --fortran

The above selects Slivr, Parallel, Python, and Fortran in the first page. From the second page just select the first item using "space"-key.
Wait while prerequisites are built.
Note: you may have to install other pre-requisities. If the pre-build fails, look at the last few lines of the log file and try to figure out what is missing.
for example you may need to add commands as below:
  • sudo apt-get update
sudo apt-get upgrade
sudo apt-get install libopenmpi1.6
sudo apt-get install libopenmpi1.6-dev
and remove:
  • sudo apt-get autoremove
    • to remove packages that were automatically installed to satisfy dependencies for some package and that are no more needed!!

Then again type the same command:
  • ./build_visit2_5_2.txt

and choose the same items mentioned above and wait while it is built.

  • at this stage you can cd to the directory as below and run visit program:
    cd visit/visit2.5.2/prebuild/src/bin/visit

  • In order to make a short cut to visit you can just go through these steps:

Open the .bashrc with an editor (vi, or gedit)

  • gedit ~/.bashrc

and go to the "# some more ls aliases" or "ALIASES" part and add this:
    • alias visit='<the directory address where you installed your visit>'

Save and exit.
    • If you use vi edit, in order to save and exit you have to just type ":x" + <Enter> .
      • In order to get the directory address you need to cd into the visit/visit2.5.2/prebuild/src/bin/visit and type pwd
to get the directory address.
Now type the below command:
    • source ~/.bashrc
to make the short-cut for your visit. Now whenever you type visit in the terminal window, the visit window will pop up.


Basically you would need to install VTK (The Visualization Toolkit which provides some of the file formats (like XML)) in order to make data-sets of our program outputs that can be visualized and analysed with any of the visualization packages that support VTK files e.g. visIt. For installing VTK you need to type these commands:
  • sudo apt-get install libvtk5.8
sudo apt-get install libvtk5-dev 
sudo apt-get install tcl-vtk

You may need to install "boost", a library for C++
  • sudo apt-get install libboost-all-dev
