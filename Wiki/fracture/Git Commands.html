Accessing and changing the Fracture-Effects repository on GitHub


Create a GitHub account
  • https://github.com/signup/free

Install git on your machine
On Ubuntu:
sudo apt-get install git-core
On Microsoft Windows:
  • http://git-scm.com/book/en/Getting-Started-Installing-Git#Installing-on-Windows

Setup git configuration
git config --global user.email "your_email@your_address.com"
git config --global user.name "your_github_username"

Email your github username to Bryan.
  • bryan.smith@callaghaninnovation.govt.nz

Create a clone of the Fracture Effects repository
git clone https://github.com/BryanSmithCI/Fracture-Effects

Check branches/master
git branch -a

Add file to local repository
git add README
git commit -m 'Added README file for git'

Check status
git status
git diff origin/master

Update github repository
git push origin master

Update local repository
git pull

Configure ssh for secure access (if needed)
cd ~/.ssh
mkdir backup
cp * backup/
rm id_rsa*
ssh-keygen -t rsa -C "your_email@your_address.com"
sudo apt-get install xclip
cd ~/Fracture-Effects/
xclip -sel clip < ~/.ssh/id_rsa.pub
ssh -T git@github.com
ssh-add
ssh -T git@github.com

From Weta repository to Github

If you ever need to do this again:

Create a clean clone
git clone gitosis@git.wetafx.co.nz:ParticleFracturing

Change the remote url
cd ParticleFracturing
git remote set-url origin https://github.com/BryanSmithCI/Fracture-Effects.git

Check the status (to make sure your are on the right track)
git remote -v

Push the code to GitHub
git push origin master

Reconfigure git
git config --global user.email "your_email@your_address.com"
git config --global user.name "your_git_username"

Uintah svn to GitHub git

The following are needed only if you want to move Uintah directly to Github. I've done this work for you.

Convert svn to git
git svn init  https://gforge.sci.utah.edu/svn/uintah -T trunk .
git svn fetch -r 49550:HEAD
git svn rebase
cp -Rf .git/refs/remotes/* .git/refs/heads/
rm -Rf .git/refs/remotes

Check branches/master
git branch -a

Configure ssh
cd ~/.ssh
mkdir backup
cp * backup/
rm id_rsa*
ssh-keygen -t rsa -C "b.banerjee.nz@gmail.com"
sudo apt-get install xclip
cd ~/fatitah/
xclip -sel clip < ~/.ssh/id_rsa.pub
ssh -T git@github.com
ssh-add
ssh -T git@github.com

Configure git
git config --global user.email "b.banerjee.nz@gmail.com"
git config --global user.name "bbanerjee"

Move to github
git remote add origin git@github.com:bbanerjee/fatitah.git
git push origin master
or
git push -f origin master

Check out repository
git clone https://github.com/bbanerjee/fatitah.git

Add file to local repository
git add README
git commit -m 'Added README file for git'

Check status
git status
git diff origin/master

Update github repository
git push origin master

Update local repository
git fetch
