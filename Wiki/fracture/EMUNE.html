"EMUNE" is a Fortran code using Peridynamics for making the fracture or crack. In order to run the programs in this directory follow the below steps:









  • docs is a directory where you can find a Word-file in which the manual for the code is explained → It is written for the users using the windows. If you are using the Linux, Ubuntu,... you need to make some adjustments.
  • src is the source where all the written programs exist. in order to run them you first need to make the opt directory:
     mkdir opt
 cd opt
 ls

    and by using the cmake you can hopefully create all the needed files including "makefile"
     cmake ../src

    and by the make command
     make

    an executable file named as: emune would be created in your opt directory.

Now you need to make another directory for running your program:
  •  mkdir runs
 cd runs
 ls







  • You need to bring the written program from the src directory to the runs directory. So while you are at the "runs" directory do as below:
  •  ls ../src → #to see which programs are in src and which one you need to copy
 cp ../src/element_test_103by42.txt element_test_103by42.txt
 cp ../src/nodes_test_103by42.txt nodes_test_103by42.txt
 cp ../src/main_input_file.txt main_input_file.txt
  • then you can run the "main" program by the following command:
  •  ../opt/emune main_input_file.txt
  • In which "emune" is the executable file made in the "opt" directory.

To visualise the out-put data, run the visIt program in the same directory (runs):
  •  ~/ParticleFracturing/EMUNE/runs$ visit
