Using Eclipse


  • Install eclipse:
sudo apt-get install eclipse-platform
sudo apt-get install eclipse-cdt

  • Use Cmake to create projects that Eclipse can understand.
    • Go to the build directory, e.g., opt.
    • Instead for cmake ../src type
cmake -G"Eclipse CDT4 - Unix Makefiles" -DCMAKE_ECLIPSE_GENERATE_SOURCE_PROJECT=TRUE -D_ECLIPSE_VERSION=3.8 ../src
On some machines the following works:
cmake -G"Eclipse CDT4 - Unix Makefiles" -DCMAKE_ECLIPSE_GENERATE_SOURCE_PROJECT=TRUE -DCMAKE_ECLIPSE_VERSION=3.8 ../src

This will create a .project and a .cproject file in the build directory and a .project file in the src directory.

  • Open the projects using Eclipse.
    • Run eclipse from the command line using "eclipse".
      • Import project using Menu File->Import
      • Select General->Existing projects into workspace:
      • Browse where your build tree is and select the root build tree directory. Keep "Copy projects into workspace" unchecked.

  • Because we are doing a out-of-source build, repeat the above procedure by going to src instead of opt (your build directory).

More information can be found at http://www.vtk.org/Wiki/Eclipse_CDT4_Generator
